# Pokédex React

Aplicación web construida con React que permite buscar Pokémon en tiempo real utilizando la PokeAPI.

## 🚀 Características

* Búsqueda en tiempo real
* Implementación de debounce
* Manejo de errores
* Interfaz simple y limpia
* Consumo de API REST

## 🛠️ Tecnologías

* React
* JavaScript (ES6+)
* CSS
* PokeAPI

## ⚙️ Instalación

```bash
git clone https://github.com/tu-usuario/pokedex-react.git
cd pokedex-react
npm install
npm run dev
```
